## 🎯 Живой демо-бот

Бот уже запущен и работает 24/7 на Render.  
 **[Попробовать бота](https://t.me/твой_бот)** 

Просто нажми `/start` и протестируй все функции.

## ✨ Возможности

- ✅ **Регистрация** с выбором роли (клиент / исполнитель)
- ✅ **Каталог услуг** с детальным просмотром
- ✅ **Создание услуг** для исполнителей (FSM)
- ✅ **Оформление заказов** с комментариями
- ✅ **Уведомления** при смене статуса
- ✅ **Админ-панель** в боте (статистика, рассылка, просмотр пользователей)
- ✅ **PostgreSQL** на Render
- ✅ **SQLAdmin** для управления БД
